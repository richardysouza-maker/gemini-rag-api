Listar Stores
Retorna todos os File Search Stores criados na sua conta Gemini.
GET /api/stores
Criar Store
Cria um novo File Search Store. É o container onde seus documentos serão indexados.
POST /api/stores

Nome amigável para identificar o store.

Deletar Store
Remove permanentemente um store e todos os documentos indexados nele.
DELETE /api/stores/:storeName

Apenas o ID, sem o prefixo fileSearchStores/

Upload de Arquivo
Envia um documento para ser indexado em um store. Suporta PDF, DOCX, TXT, JSON, CSV, HTML, MD e mais.
POST /api/upload
📄
Arraste um arquivo ou clique para selecionar

Metadados permitem filtrar documentos específicos nas consultas.

Listar Documentos
Lista todos os documentos indexados em um store específico.
GET /api/upload/:storeName
Deletar Documento
Remove um documento específico de um store.
DELETE /api/upload/:storeName/:docId
Chat RAG

POST /api/chat
Streaming (SSE)

Limpar Sessão
Remove o histórico de conversa de uma sessão da memória do servidor.
DELETE /api/chat/session/:sessionId
